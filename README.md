# Cheese Chase 3D – v2 (Multi-file, Scene Graph, Gameplay)

- Proper composite meshes for **Mouse**, **Cat**, and a low‑poly **Cheese Wedge**.
- **Cheese collection**: disappears on pickup.

- Fully **multi‑file**: Camera, Shader, Mesh, Node, SceneGraph, GameState, Systems, etc.
- Includes **THIRD_PARTY_NOTES.txt** per requirements.
